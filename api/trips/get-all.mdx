## Endpoint

```
GET /trips
```

Retrieve a list of all trips in the system.

## Response

<ResponseField name="trips" type="array">
  Array of trip objects
  
  <Expandable title="Trip Object">
    <ResponseField name="id" type="integer">
      Unique identifier for the trip
    </ResponseField>
    
    <ResponseField name="user_id" type="integer">
      ID of the user who owns the trip
    </ResponseField>
    
    <ResponseField name="title" type="string">
      Title of the trip
    </ResponseField>
    
    <ResponseField name="shelter" type="object">
      Geographic point representing the shelter location (PostGIS POINT type)
    </ResponseField>
    
    <ResponseField name="arrive_date" type="string">
      Arrival date for the trip
    </ResponseField>
    
    <ResponseField name="leave_date" type="string">
      Departure date for the trip
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash
curl --request GET \
  --url https://api.maytravel.com/trips
```

## Status Codes

- **200** - Success: Returns array of all trips
- **500** - Server Error: An error occurred while fetching trips
